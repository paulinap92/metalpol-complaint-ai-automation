# Metalpol Complaint AI Automation

## Overview

This repository contains a conceptual proposal for automating the complaint handling process at Metalpol, a fictional automotive manufacturing company.

The project focuses on:
- process analysis,
- AI-assisted automation,
- enterprise system integration,
- operational workflow optimization.

The proposed solution combines:
- Microsoft Graph API,
- SAP ERP integration,
- JIRA automation,
- AI-assisted complaint processing,
- human-in-the-loop validation.
